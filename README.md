# Item Category

A TypeScript Node.js project for managing item categories.

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Usage

```bash
npm start
```

## Development

```bash
npm run dev
```

## Building

```bash
npm run build
```

## Project Structure

```
src/
├── index.ts
├── types/
└── utils/
```
